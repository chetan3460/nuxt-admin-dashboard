# Nexus Dashboard

A modern, high-performance admin dashboard built with Nuxt 3, Tailwind CSS, and TypeScript.

## Features

- **Modern Stack**: Built with Nuxt 3, Vue 3, and TypeScript
- **UI Components**: Custom UI library built on top of Tailwind CSS
- **Charts**: Integrated ApexCharts for beautiful data visualization
- **Responsive**: Fully responsive design with mobile support
- **Theming**: Dark/Light mode support with customizable themes
- **Icons**: Lucide icons integration

## Tech Stack

- **Framework**: [Nuxt 3](https://nuxt.com)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **State Management**: [Pinia](https://pinia.vuejs.org)
- **Charts**: [ApexCharts](https://apexcharts.com)
- **Icons**: [Lucide](https://lucide.dev)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

### Build

```bash
# Build for production
npm run build
```
